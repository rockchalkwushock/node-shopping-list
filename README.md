# node-shopping-list
Basic Shopping List App using Node/Express.js, RESTful API, &amp; CRUD.

## Technologies Utilized:
  -  Node.js
  -  Express.js
  -  Express Body Parser
  -  Nodemon
  -  Mocha
  -  Chai / ChaiHttp
  -  Istanbul
